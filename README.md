# Currency Converter

Простой и лёгкий веб-конвертер валют с использованием открытого API для получения актуальных курсов. Проект представляет собой статический фронтенд (HTML/CSS/JS) и подходит для деплоя на GitHub Pages, Vercel или Netlify.

---

**Описание**

- **Название:** Currency Converter
- **Цель:** Быстрое и удобное конвертирование сумм между валютами с использованием актуальных курсов.
- **Ключевые особенности:**
  - Конвертация валют в реальном времени
  - Поддержка большого числа валют (через ER-API)
  - Кастомные выпадающие списки с флагами
  - Кнопка "Swap" для мгновенного обмена валют
  - Адаптивный интерфейс

---

**Структура проекта**

- `index.html` — главная страница и разметка
- `style.css` — стили проекта
- `script.js` — логика приложения, загрузка курсов, конвертация
- `public/` — статические ресурсы (иконки, SVG и т.д.)
- `README.md` — этот файл

**Технологии**

**Запуск локально**

Проект — статический, поэтому достаточно открыть `index.html` в браузере. Для более корректного тестирования (и отсутствия проблем с CORS при локальной разработке) рекомендуется использовать локальный HTTP-сервер.

Примеры команд (PowerShell / Windows):

```powershell
# 1) Если установлен Python 3:
py -3 -m http.server 8000
# Затем откройте в браузере: http://localhost:8000

# 2) Если установлен Node.js (используя пакет http-server):
npm install -g http-server
http-server -p 8000
# Затем: http://localhost:8000
```

**Конфигурация и использование**

- По умолчанию приложение использует API: `https://open.er-api.com/v6/latest/USD`.
- При загрузке страницы происходит запрос курса и заполнение списков валют.
- Введите сумму в поле `Amount`, выберите валюты "From" и "To" — приложение покажет результат и ориентировочный курс.
- Кнопка `Swap` меняет выбранные валюты местами.

---

**Ключевые фрагменты кода**

- Загрузка курсов:
```javascript
const API_URL = "https://open.er-api.com/v6/latest/USD";
async function loadCurrencies() {
  const res = await fetch(API_URL);
  const data = await res.json();
  if (data.result !== "success") return;
  rates = data.rates;
  // заполнение select'ов и UI
}
```

- Конвертация:
```javascript
function convert() {
  const amount = parseFloat(amountInput.value) || 0;
  const rate = rates[to] / rates[from];
  const result = (amount * rate).toFixed(2);
  resultEl.textContent = result;
}
```

**API и внешние ресурсы**

- ER-API (Exchange Rate API): https://www.exchangerate-api.com/
